Easy Linked Open Data

Lodlab allows everyone to publish 5-star Linked Open Data starting with a simple spreadsheet.

Join the Open Data Revolution

Start with CSV

Lodlab converts CSV to linked data. It takes care of all the technical details so you can just focus on your data.

Its Your Data, You Keep It!

Your data and all of the linked data files are held in your own GitHub repository and published using GitHub Pages. The lodlab.com server simply redirects page requests to the GitHub Pages site. This gives you total control over the data all the time.
